# Myco Style Shadow Pro

**这不仅仅是一个算法助手，它是你的代码双胞胎。**

`myco.skill` 的核心价值在于 **极致的风格模仿 (Style Shadowing)**。它通过深度扫描你本地的代码库，提取你的编程习惯（代码 DNA），从而生成完全符合你个人风格的算法代码，彻底消除“AI 感”。

## 🎯 核心价值：代码 DNA 克隆

不同于普通的 AI 编程助手，本 Skill 会强制执行以下逻辑：
- **微观习惯捕捉**：精确到你对空格的使用、大括号的位置以及 `i++` 还是 `++i`。
- **算法偏好学习**：你是习惯用链式前向星还是 `std::vector`？它是习惯用全局变量还是局部传递？它会通过 `grep` 学习你的所有选择。
- **致敬“烂习惯”**：如果你喜欢压行或者不爱写注释，它会严格遵守，绝不自作聪明地“优化”你的代码结构。

## ✨ 其它功能

- **主动确认**：在执行前询问爬题需求及任务范围。
- **模板优先**：自动识别并应用你的 `template.cpp`。
- **自动验证**：抓取样例并在本地编译运行，确保逻辑正确。
- **人类思维模拟**：以第一人称撰写 `THOUGHTS.md`，记录解题时的心路历程。
- **跨工具联动**：支持一键生成题目讲解 PPTX。

## 📂 目录结构

```text
solutions/
└── [Platform]/
    └── [ProblemID]/
        ├── solution.cpp    (你的“克隆”代码)
        ├── sample.in       (样例输入)
        ├── sample.ans      (样例输出)
        ├── THOUGHTS.md     (你的“心路历程”)
        └── presentation.pptx (讲解PPT)
```
